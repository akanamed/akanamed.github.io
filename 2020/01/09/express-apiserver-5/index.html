Node.js process

Node.js의 process 객체는 global이다.
즉, require() 를 사용하지 않고, 접근할 수 있으며, 현재 프로세스에 대한 정보와 제어를
할 수 있다.

process 객체 중 이번 포스팅에서 다루는 건 env 환경 변수인데, dotenv 패키지를 이용해
프로젝트에서 사용할 환경 변수들의 정보를 .env 파일에 모아서 관리한다.
